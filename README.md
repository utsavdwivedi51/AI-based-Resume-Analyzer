# AI-based-Resume-Analyzer
An intelligent ATS (Applicant Tracking System) checker that analyzes 
resumes against job descriptions using NLP algorithms.

## Features
- PDF & Text resume upload support
- Real-time keyword matching analysis
- ATS compatibility score (0-100%)
- Skill detection for 30+ technical skills
- PDF report export functionality
- 100% client-side processing (privacy-first)

## Technologies
- JavaScript (ES6+)
- HTML5 & CSS3
- PDF.js for document parsing
- jsPDF for report generation
- Natural Language Processing algorithms
